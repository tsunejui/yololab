🆚 判別式 AI vs 生成式 AI

兩種 AI 技術在本系統中的職能與特性對照

比較項目 🚁 判別式 AI 🖥️ 生成式 AI
主要任務 分類、偵測、計數 文字生成、推理、決策建議
代表模型 YOLO、ResNet、ViT、SAM GPT-4o、Claude 3.5、Llama 3
輸入資料 影像、影片、感測器讀數 結構化資料 + 知識庫 + Prompt
輸出形式 邊界框、類別、追蹤 ID、數量 Markdown 報告、建議書、PDF
部署位置 邊緣端（無人機晶片 / Jetson） 雲端 / 後端伺服器
推論延遲 ~8 ms / frame（即時） 數秒至數十秒
算力需求 中等，可邊緣化（TensorRT 加速） 大型 GPU 集群或 LLM API
學習方式 標註資料 + 監督式學習 大規模預訓練 + RAG + Prompt
系統中角色 即時感知與計數（眼睛） 推理分析與報告生成（大腦）
代表工具 OpenCV、Ultralytics、TensorRT LangChain、CrewAI、ChromaDB
